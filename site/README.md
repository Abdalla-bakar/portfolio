# Abdalla Kombo Bakar — Portfolio

Static HTML/CSS/JS portfolio site.

## Local preview
Open `site/index.html` directly, or serve the folder:
```bash
npx serve site
```

## Deploy

### Vercel
The repository ships with `vercel.json` at the root. Import the repo in Vercel — no build command required, output directory is `site/`.

### Render
The repository ships with `render.yaml` at the root. Create a new "Blueprint" on Render pointing at this repo; Render will create a static site serving `./site`.
